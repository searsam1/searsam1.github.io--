String Formatting

String formatting makes a world of difference.

Instead of adding string together it formats the string based on the code given.

Old Way

One can use the modulo % operator in a string to format it. 

import math
print('The value of e is approximately %5.5f.' % math.e)
# The value of e is approximately 2.71828.

message = "hello world!"
print('Message : %s' % message)
# Message : hello world!

Using string's format() method

Strings in python come with the str.format() method.

{ } are used to enclose a variable. Instead of putting the variable itself in the { } , the index of the variable passed in goes there, or, one can skip the index if the arguments are the same length and match up. Argument 0 will be mapped to the 0th { } , argument 1 will be mapped to the 1st { } , and so on.

"{} {} {}".format(1, 2, 3)

'1 2 3'

Format Strings (f strings)

Python 3.6 introduced format string literals, an easy way to format strings using f"{something} {goes_here}"

Technically those are string literals. Formatted string literals are more precise and give more arguments. f"{math.pi:.3f}".

Giving a number after the semi colon will make it n characters wide at minimum. And the f means it will be a float. 


first_name, last_name = "john", "sullivan"
full_name = f"{first_name.title()} {last_name.title()}"
print(full_name)
# John Sullivan

Conversion Meaning
'd'

Signed integer base 10. [Source]

f"{5568547:d}" -> '5568547'

'o'

Signed octal [Source]

f"{256:o}" -> '400'

'x'

Signed hexadecimal (lowercase). [Source]

f"{255:x}" -> 'ff'

'X'

Signed hexadecimal (upper). [Source]

f"{255:X}" -> 'FF'

'e'

Floating point exponential format (lowercase). [Source]

f"{255:e}" -> '2.550000e+02'
'E'

Floating point exponential format (uppercase). [Source]

f"{255:E}" -> '2.550000E+02'
'f'

Floating point decimal format. [Source]

f"{255:f}" -> '255.000000'
'F'

Floating point decimal format. [Source]

f"{255:F}" -> '255.000000'
'g'

Floating point format. Uses lowercase exponential format if exponent is less than -4 or not less than precision, decimal format otherwise. [Source]

f"{255.777676:g}" -> '255.778'

'G'

Floating point format. Uses lowercase exponential format if exponent is less than -4 or not less than precision, decimal format otherwise. [Source]

f"{255.777676:G}" -> '255.778'

'c'

Single character (accepts integer or single character string). [Source]

f"{100:c}" -> "d"

'r'

String (converts any Python object using repr()). [Source]
's' String (converts any Python object using str()) .[Source]
'a' String (converts any Python object using ascii()).[Source]
'%'

No argument is converted, results in a '%' character in the result. [Source]

f"{1:%}" -> '100.000000%'
