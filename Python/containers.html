Containers

A container is an object that can hold other objects. Python has a vast collection of containers ranging from dictionaries to lists to tuples.

Lists are easy to work with in python and come in handy often, especially with list comprehensions. Lists are created by using square brackets to encapsulate the list and commas to separate the items. 


lst = [1, 2, 3, 4]
lst2 = ["strings"]
lst3 = [["another list"], lst, lst2]

Elements are located by index also using square brackets. They always start at zero at go to the length of the list - 1. The range function starts at zero and goes to the end parameter - 1, making it perfect for iteration and lists. 


lst = ["a", "b", "c"]

for i in range(len(lst)):
    print(i, lst[i])
# 0 "a"
# 1 "b"
# 2 "c"

# better to use enumerate(), but 
# trying to keep it simple for this example

Assigning a value to a list is easy as well. Reference the list and the index and set it equal to whatever value.
lst[0] = "1"
