Containers

A container is an object that can hold other objects. Python has a vast collection of containers ranging from dictionaries to lists to tuples.

What is a list

Lists are easy to work with in python and come in handy often, especially with list comprehensions. Lists are created by using square brackets to encapsulate the list and commas to separate the items. 


lst = [1, 2, 3, 4]
lst2 = ["strings"]
lst3 = [["another list"], lst, lst2]

Elements are located by index also using square brackets. They always start at zero at go to the length of the list - 1. The range function starts at zero and goes to the end parameter - 1, making it perfect for iteration and lists. 


lst = ["a", "b", "c"]

for i in range(len(lst)):
    print(i, lst[i])
# 0 "a"
# 1 "b"
# 2 "c"

# better to use enumerate(), but 
# trying to keep it simple for this example

Assigning a value to a list is easy as well. Reference the list and the index and set it equal to whatever value.
lst[0] = "1"

Dictionaries in python have a structure where there are key value pairs that match each other. There can only be one value per key, because if there was more python would not know which value to get.

They are created with { } and the key-value pairs are separated by a colon. To access a value, use dic[key]. 


d = {"name" : "john smith"}
print(d["name"])
# "john smith"

To loop over a dictionary the items() method must be used to gain access to the keys and values. By default, only the keys are available when iterating. for k,v in d.items() is the norm, k and v being arbitrary names standing for key and value. 


apartment_directory = {
    "Manny" : 101,
    "Carla" : 102,
    "Boone" : 103
}

for k, v in apartment_directory.items():
    print(f"Name: {k} | Apartment #{v}")    

# Name: Manny | Apartment #101
# Name: Carla | Apartment #102
# Name: Boone | Apartment #103

print(
    apartment_directory["Manny"]
)
# 101

Tuples are static containers meaning they cannot be altered after creation. Tuples are made with ( ) and the values are separated by commas. 


tup = (1, 2, 3)
tup[0] = 2

To make changes to a tuple, create a copy of that tuple with them. It is probably better to use a list at that point, but it is possible. They can still be iterated over and have some of the same uses as lists.
