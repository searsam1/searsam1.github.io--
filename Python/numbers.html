Numbers in Python

Python has a every basic number type including floats and integers. If an object can be converted to an integer use the int() function. Use the float() function to convert to floats. 


x = 5
y = 10
z = 5.5
print(int(z))
# 5
print(float(x))
# 5.0
print(float(y))
# 10.0

int Methods

Python comes ready with math functions preloaded like add, minus, divide and subtract.

Some common methods include

  • abs()
  • float()/int()
  • str()
  • round()
  • pow() 


x = 5
print(abs(x), float(x), str(x), 
    round(x), pow(x,2), sep="\n"
    )
# 5
# 5.0
# "5"
# 5
