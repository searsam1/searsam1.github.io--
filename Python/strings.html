Strings In Python

What is a string?

A string in a programming language is a string of letters. Imagine a string going through a bunch of letters like a clothes line. Each time a new clothing item is clipped on the line gets heavier. Now, instead of those being clothing items hung up, it is letters or characters in the alphabet. This paragraph is a string because it is a collection of letters and spaces formed to make words.

To create strings in python "string" (double quotes) or 'string' (single quotes) can be used. Other languages, like C++, have a feature where only single characters can go into single quotes, and only strings can go into double quotes. Therefore, it is good practice to use double quotes " with strings in python, even though it is not required.

Characters

Strings are composed of characters. How do these characters get converted to machine code so that they can be read by the computer? The ASCII table solves this by mapping characters to numbers. For example, 'A' maps to 65, 'B' maps to 66 'C' maps to 67 and so on until 'Z'

So really, strings are just a collection of numbers that have a special meaning to them.

Raw Strings

Raw strings treat the backslash \ as a literal character. Printing "\n" will normally give a newline, but with a raw string it treats it as a literal "\n". 

        
# Prints a newline
print("hey \n hey")
# hey
#  hey

# Prints literal \n
print(r"hey \n hey")
# hey \n hey

Strings are Arrays of Characters

Single characters can be accessed through the [ ] operator, with the index specifying which location letter is in. If s = "hello", then s[0] == "h". They are zero-indexed like normal arrays.
