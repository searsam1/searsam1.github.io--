What Are Classes in Programming?

Classes are a way to abstract objects so that a computer can understand what the object is. In other words, classes help to make different objects in python that are similar to one another but still separate objects.

Are good example is to think of crayons. They all do the same thing but some are slightly different than others. Some are yellow and some are green, but they all have a color which is an attribute of all crayons. Every crayon can write as well: this is a function of all crayons. Classes usually have attributes and functions which can be modified or run. These methods(functions) and attributes can be further divided into public and private parts. Public being it can be accessed anywhere, private being it can only be accessed by the class.

Classes in Python

To define a new class in python, use the class keyword, followed by the name of the new class, followed by a colon: class Class:. This is defining the class itself and therefore does not create any instances of it.

An instance is when the class is used to create a new object in the program, which is done by "calling" the class and usually setting it equal to a variable. 


# Define new Crayon class
class Crayon:
    ...
# Create new instance named crayon
crayon = Crayon()    

# "..." just tells python we 
# will define the class later

Instance Variables

An instance variable is connected to the instance that created it. They are basically variables passed into the function when the object is created.

Python uses the special __init__() method to define instance variables. It is usually placed at the beginning of the class. 


class Crayon:
    def __init__(self, color:str):
        self.color = color

green_crayon = Crayon("green")    
print(green_crayon.color)
# 'green'

Instance variables can also be accessed throughout the class by referencing self. Self is an arbitrary variable name; it represents the underlying instance inside the instance itself. 


class Crayon:
    def __init__(self, color:str):
        self.color = color

    def print_color(self):
        print(self.color)

    # using "crayon" instead of self 
    # to better illustrate the point    
    def change_colors(crayon,color):
        crayon.color = color

green_crayon = Crayon("green")
green_crayon.print_color()
# Ouputs -> green
green_crayon.change_colors("yellowgreen")
green_crayon.print_color()
# "yellowgreen"
