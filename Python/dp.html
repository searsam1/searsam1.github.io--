Python Double Palindromes

What is a palindrome?

Palindromes are words that stay the same when they are reversed.

Ex. otto, bob

A double palindrome is where there are numbers and letters in the string. Separate the two and if they are both palindromes it is a double. If it is one or the other it is a normal palindrome.

Ex. cb77c, ccc888

cb77c = cbc, 77

Check Palindrome in python 


def check_pal(s):
    return s[::-1] == s

def palindrome_set(lst):
    scores = []
    for word in lst:
        chars = "".join(i for i in word if i.isalpha())
        nums = "".join(i for i in word if i.isnumeric())
        score = check_pal(nums) + check_pal(chars)
        scores.append(score)
    return scores
