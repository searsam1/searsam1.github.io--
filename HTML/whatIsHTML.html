What is HTML?

Hyper Text Markup Language, or HTML for short, is the structure of the modern web. It uses tags, which are variable letters wrapped in angle brackets (less than and greater than signs).

For example, the <p> tag is the "Paragraph" tag and the <i> tag is the "italics" tag. These tags change the structure of the website; <p> basically adds a newline by default to signify new paragraphs. Obviusly tags do not have to modify the structure, like with the <i> tag (makes the text italic) or with the anchor tag <a> , which makes hyperlinks. Hyperlinks are links that navigate the web browser from one place to another.

In order to create a website it is imperative to know html. While it is possible to use a website creator service , it is much easier to learn HTML and create a website because it is more reproducible and creates better websites.

A topic of much controversy, HTML is a structuring language over a coding language. The point of HTML is to add organization and structure to the website, and websites would be much more sporadic in their source code. HTML gives a baseline to expect for the entire web.

According to the University Of Washington, Tim Berners-Lee wrote the first version of HTML in 1993. Version 2 came out in 1995, followed by 3 and 4 both in 1997. HTML 5 is the most recent.

Here is a basic example of what HTML looks like: 


<!DOCTYPE html>            
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Website</title>
    <link rel="stylesheet" href="./style.css">
    <link rel="icon" href="./favicon.ico" type="image/x-icon">
  </head>
  <body>
    <main>
        <h1>Welcome</h1>  
    </main>
	<script src="index.js"></script>
  </body>
</html>
