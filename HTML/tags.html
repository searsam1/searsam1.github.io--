HTML - Tags

Tags are the main component to html syntax. A tag is simply a character or word surrounded by angle brackets <like this>. There are normal start tags and then end tags and the text goes between the two. End tags have a / to separate them from start tags. <span> content </span> Tags are used to create elements, which are the different "parts" of an html document.

The top of the elements is <html>, also known as the root element. This is because all elements must stem from this element.

There are different types of tags that serve different functions. The meta tags are used for metadata about the page, like fonts to load and what language the document is in. These tags include <base>, <head>, <link>, <meta>, <style> and <title>. The <head> tag is where a lot of important links are with links to stylesheets, scripts and other important metadata. <link> talks to other documents to get external information like CSS and apply it to the current document.

The body of the page is contained in the<body> tag, which is the base of all body elements. Most websites have a header, footer and a body. The meat of the website comes from the body tag. In the body, there are two main types of sections; content sections and text sections. <nav> is an example of a content body tag. It represents a section of the page with navigation links. <h6> turns the text bigger and into a header-like format. The smaller the number next to h, the bigger the text will be (i.e. <h6> will be smaller than <h5>). Some examples of text content tags are <div>, <ul> and <li>. <div> doesn't actually do anything on its own. It is more of a generic container. <ul> stand for unordered list, and <li> being a list item inside that list.

There are inline text style tags as well. The <a> or anchor tag combined with href creates clickable links to other web pages. The <b> or bold tag makes text pop by making it darker and thicker. Href is an attribute of the <a> tag. Attributes attribute themselves to the tag and make the tag act a certain way, given the correct information. They go after the tag name and are assigned values like variables.
