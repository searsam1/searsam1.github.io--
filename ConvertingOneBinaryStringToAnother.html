Converting Binary Strings to Each Other

The overall goal is to find the #number of swaps needed to convert one of the strings to the other.

The challenge is to find which positions in the binary string do not match (all strings will be equal in length). Take the number of positions that are different and divide that by 2.

Original Challenge 

function minSwaps(s1, s2) {
	var count = 0;
  	for (const i in s1){
	  if (s1[i] != s2[i]){
		count += 1;
	  }
	}
    return count / 2;
}
minSwaps("1001","0110")
// 2

minSwaps("100110011001","011001100110")
// 6
        

def min_swaps(s1, s2):
    res = sum(map(lambda x: x[0] != x[1], zip(s1, s2)))
    return res // 2


def min_swaps(s1, s2):
    s1, s2 = int(s1,2), int(s2,2)
    return (s1 ^ s2).bit_count() // 2
